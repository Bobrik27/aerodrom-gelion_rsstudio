# Проектный Бриф: "Аэродром Гелион"

**Дата начала:** 2025-11-03
**Статус:** В разработке

## 1. Цель проекта

Создание современного, быстрого, масштабируемого и SEO-оптимизированного сайта для аэродрома "Гелион". 

**Ключевая аудитория:** Частные владельцы самолетов, пилоты, потенциальные клиенты.

**Планируемый функционал:**
- Информационная часть (описание услуг, новости).
- Привлечение новых клиентов (формы обратной связи, калькуляторы).
- Интеграция сторонних сервисов (бронирование отелей, парковки).
- Полезные виджеты (справки о погоде).

---

## 2. Технологический стек

- **Фреймворк:** Next.js (с App Router)
- **Язык:** TypeScript
- **Стилизация:** Tailwind CSS
- **Контроль версий:** Git / GitHub

---

## 3. Структура проекта

- `airport-proj/`
  - `src/`: Основной исходный код приложения.
    - `app/`: Роутинг и страницы. `layout.tsx` - главный шаблон, `page.tsx` - главная страница.
    - `components/`: Переиспользуемые React-компоненты (кнопки, шапка, подвал).
    - `lib/`: Вспомогательные функции, не связанные с UI (например, работа с датами).
  - `public/`: Статические файлы (изображения, иконки, шрифты).
  - `docs/`: **Эта документация.** Содержит бриф и журнал разработки.
  - `analysis/`: Исследования, анализ конкурентов, заметки.
